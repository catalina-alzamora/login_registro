# Template de examen: Java / Spring

### Plantilla de Proyecto Spring Boot (Java) para Exámenes de Skillnest

Este template contiene un proyecto base que incluye todas las dependencias y properties necesarias para usar como punto de partida en tu examen.

> Recuerda cambiar las properties de `username` y `password` para poder conectarte correctamente a la base de datos.

> Además, deberás cambiar la `url` de la conexión a la DB en caso de que elijas nombrarla de otra forma.
